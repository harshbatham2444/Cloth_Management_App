# 👕 AI Cloth Size Measurement

## AI-Powered Virtual Fitting Assistant

This project uses Artificial Intelligence and Computer Vision techniques to estimate clothing size through a webcam. The application detects body landmarks in real time and provides clothing size recommendations to improve the online shopping experience.

## Features

- Real-time body tracking using webcam
- AI-based body landmark detection
- Automatic body measurement estimation
- Clothing size prediction (S, M, L, XL, XXL)
- User-friendly graphical interface
- Fast and lightweight implementation

## Technologies Used

- Python
- OpenCV
- MediaPipe
- PyQt5

## Installation

```bash
pip install opencv-python mediapipe PyQt5
```

## Run the Application

```bash
python cloths.py
```

## Applications

- Virtual fitting assistance
- Online apparel size recommendation
- E-commerce integration
- Personalized shopping experience
